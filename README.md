📋 Task Tracker

Трекер задач на Go с консольным и веб-интерфейсом.

Возможности

- ✅ Добавление задачи с названием и описанием
- ✅ Просмотр списка задач
- ✅ Отметка задачи как выполненной
- ✅ Удаление задачи
- ✅ Поиск по задачам (консольная версия)
- ✅ Цветной интерфейс в терминале
- ✅ Веб-версия в браузере с тёмной темой
